מוסיף תיבות גלילה לבחירת רחוב לפי עיר בישראל עבור כל טופס עם שדות עיר-רחוב (או עיר בלבד) (Woocommerce, טופס יצירת קשר 7, Elementor וכו').

== תיאור ==

שפרו את חווית המשתמש בישראל עם בחירת עיר ורחוב בדיוק ובקלות לטפסים! תוסף זה מוסיף תיבת גלילה לכל שדה עיר ורחוב (או רק עיר) שתגדירו. תואם לבוני טפסים פופולריים כמו WooCommerce, Contact Form 7, Elementor ועוד. מייעל את התהליך עבור המשתמשים בישראל בכך שהוא מאפשר להם לבחור עיר מרשימה ורחוב על בסיס העיר.

נתוני הערים והרחובות המתאימים להן נמשכים אוןליין מה API של data.gov.il 

== מאפיינים ==

חווית משתמש משופרת: תפריט נפתח אינטואיטיבי לבחירת עיר מרשימה ואז רחוב לפי עיר.

דיוק: מפחית שגיאות בכתובת

תאימות: עובד בצורה חלקה עם בוני טפסים שונים שנמצאים בשימוש נרחב באתרי וורדפרס

== למי מיועד תוסף זה? ==

תוסף זה אידיאלי עבור כל בעל אתר וורדפרס בישראל המשתמש בטפסים לאיסוף פרטי משתמש, לרבות חנויות מקוונות (WooCommerce), טפסי יצירת קשר (טופס צור קשר 7), ובוני אתרים עם פונקציונליות טפסים (Elementor).

== התקנה ==

יש להעלות את תיקיית הפלאגין לספריית /wp-content/plugins/, או להתקין ישירות דרך תפריט התוספים של וורדפרס.
הפעלו את התוסף דרך תפריט 'תוספים' בוורדפרס.
הגדרו את הגדרות הפלאגין לפי הצורך.

== הגדרות ==

1. עברו לדף הגדרות התוסף מתוך הגדרות - בחירת ערים ורחובות בישראל – ATR
2. ב-2 שדות ההגדרות הראשונים כתבו את ברירת המחדל של מזהה css (ID) של שדות עיר ורחוב. המזהים האלו (של שדות עיר ורחוב) ישמשו בכל טופס המכיל אותם.
3. בשדה ההגדרות השלישי כתבו מזהה עמוד (page ID) מופרד בפסיק, מזהה קלט עיר, מזהה קלט רחוב (לדוגמה: 123,city-choice,street-choice בדוגמא זו הכיוון מתהפך בגלל העברית, בפועל מזהה העמוד ראשון). אפשר להגדיר מספר עמודים שונים כשלכל אחד מהם מזהה שדות שונים, כל אחד בשורה חדשה. המזהים הללו ישמשו בכל עמוד שהוגדר. בעמודים המוגדרים באפשרות זו (השלישית) תהיה עדיפות לשדות אלו. (כלומר, אם יהיו בעמוד טפסים נוספים עם שדות ברירת מחדל מאפשרות 1,2 בהגדרות התוסף יתעלם מהם)
4. ודאו שמזהי שדות הטפסים מוגדרים כהלכה.

=== דוגמה: ===
מזהה שדה ברירת מחדל עבור ערים: <code>city-choice (שדה ראשון בהגדרות)</code>
מזהה קלט ברירת מחדל עבור רחובות: <code>street-choice (שדה שני בהגדרות)</code>
<strong>שדה שלישי בהגדרות:</strong><pre>1112,contact-city,
2274,some-city,some-street
422,billing_city,billing_address_1</pre>

שימו לב בדוגמה שבשדה 3 שורה 1 לא מוסיפה רחוב, היא מיועדת לטפסים שיש להם רק שדה עיר.
השורה האחרונה אופיינית לטופס הצ'קאאוט בווקומרס. (מזהה 422 באתר לדוגמה)
לפי הגדרות אלו, התוסף יגדיר את תיבות הגלילה עבור המזהים שצוינו בדפים אלו ולא את ברירת המחדל.

== שאלות נפוצות ==

= מדוע התוסף הזה מיועד לישראל בלבד? =

אתרי ממשלת ישראל מספקים API לערים ורחובות המשמשים תוסף זה. אם יש לך פרטים על ממשקי API של מדינות אחרות, נא עדכנו אותי ואנסה ליישם גם אותו.

= באיזה סוג של טפסים ניתן להשתמש בתוסף זה? =

ניתן להשתמש בו בכל טופס וורדפרס כל עוד שדות העיר (ו/או הרחוב) מקבלים מזהה (ID) css ייחודי לכל אחד שאותם ניתן להגדיר בהגדרות הפלאגין.
